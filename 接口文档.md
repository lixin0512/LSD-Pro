# 接口文档

### 一、统一格式

```json
{
  "success": true,
  "code": 200,
  "message": "成功",
  "data": {
    ...
  }
}
```

**在下面我直接写data里的内容，自行添加后返回给前端**

### 二、状态码

```text
400:语义有误
401:"服务器认证失败"
403:"服务器拒绝访问"
404:"地址错误"
500:"服务器出错"
502:"服务器无响应"
```
